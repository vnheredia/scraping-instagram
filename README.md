# 📊 Instagram Scraping & Analysis

## 📌 Descripción

Este proyecto realiza web scraping de perfiles públicos de Instagram utilizando Python y la librería requests, extrayendo información del perfil y publicaciones.

## ⚙️ Tecnologías

* Python
* requests
* CSV

## 🚀 Funcionalidades

* Extracción de datos del perfil (usuario, biografía, seguidores)
* Obtención de publicaciones recientes
* Cálculo de estadísticas (likes, comentarios, engagement)
* Exportación de datos a CSV

## 📊 Métricas calculadas

* Promedio de likes
* Promedio de comentarios
* Engagement rate

## ⚠️ Limitaciones

Instagram implementa mecanismos de seguridad que pueden bloquear solicitudes automatizadas.

## ▶️ Ejecución

```bash
python scraper.py
```

## 📁 Output

Archivo CSV con datos de publicaciones
